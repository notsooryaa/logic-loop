### Getting Started

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   ionic serve
   ```

## Tech Stack

- **Framework:** Ionic
- **Frontend:** Angular
